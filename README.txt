Dans la mesure où, le projet vous affiche une erreur, c'est qu'il manque peut être, AudioKit. Nous vous invitons à le télécharger au lien suivant :

-> http://github.com/audiokit/AudioKit/releases/download/v4.5.5/AudioKit-iOS-4.5.5.zip

Puis à rajouter le fichier AudioKit.framework & AudioKitUI.framework au projet dans XCode

Ensuite vous trouverez ci dessous le lien pour accéder à la base de données :

https://console.firebase.google.com/u/0/project/seizethemuse/database/seizethemuse/data

Il vous faut par contre un gmail pour accéder à la base
